Synapse – Privacy Policy
Last Updated: 13 March 2026

1. Introduction
This Privacy Policy explains what data Synapse (“the Bot”) collects, how it is used, and how it is protected.
By using Synapse, you agree to this Privacy Policy.

2. Data We Collect
Synapse only collects the minimum amount of data required to operate:

A. Server Data
- Server IDs
- Channel IDs (for configuration)
- Role IDs
- Server settings and Synapse configuration

B. User Data
- User IDs (for commands, moderation, and internal systems)
- Username/Tag (logged temporarily when needed)
- Permissions data (to verify command access)

C. Temporary or Automatic Data
- Command usage logs (non-sensitive)
- Error logs used for debugging
- Temporary cache data

This data is never sold, shared, or used outside operational requirements.

3. Data We DO NOT Collect
Synapse does NOT collect or store:
- Direct messages (DMs)
- Message content (except when required for a specific command)
- Personal information
- Email, phone numbers, or passwords
- Payment or financial information
- Sensitive or identifying personal data

4. How We Use the Data
Data is used for:
- Running Bot features
- Maintaining server-specific settings
- Moderation systems (if enabled)
- Debugging errors and improving stability
- Preventing abuse or rule violations

5. How Long Data Is Stored
- Server settings remain until the server removes the Bot.
- Temporary logs are automatically cleared regularly.
- Blacklist/anti-abuse data may be stored indefinitely for security.

6. Who Can Access the Data
Only the Bot owner and authorized developers (if any) can access limited logs or stored data.
No data is shared with third parties.

7. Data Security
We use reasonable security practices to protect stored data from unauthorized access.
However, no system can guarantee 100% security.

8. Your Rights
Server owners may:
- Request removal of stored server configuration data.
- Request information about what is stored for their server.
- Remove the Bot at any time, automatically deleting most active data.

9. Children’s Privacy
Synapse does not collect information knowingly from children under 13.
Users must comply with Discord age requirements.

10. Changes to This Privacy Policy
This policy may be updated at any time without notice. Continued use of Synapse means acceptance of the updated policy.
